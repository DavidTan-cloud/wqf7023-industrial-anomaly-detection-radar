# WQF7023 Industrial Anomaly Detection Using S-RADAR

## Overview

This repository contains the implementation of an AI-based anomaly detection framework for industrial time-series data using the S-RADAR framework.

### Datasets
- MIMII
- SMAP
- ETT

### Models
- LSTM-AE
- SISVAE
- RANCoder
- LADOP
- SMDA-Net

### Framework
- S-RADAR

### Evaluation Metrics
- Precision
- Recall
- F1 Score
- Accuracy
- AUC-ROC
