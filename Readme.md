# WQF7023 Industrial Anomaly Detection Using S-RADAR

## Overview

This repository contains the implementation of an AI-based anomaly detection framework for industrial time-series data based on the S-RADAR framework.

### Datasets
- MIMII (In progress)
- SMAP
- ETT (In consideration)

### Models
- LSTM-AE
- SISVAE
- RANCoder
- LADOP
- SMDA-Net

### Framework
- Based on S-RADAR

### Evaluation Metrics
- Precision
- Recall
- F1 Score
- Accuracy
- AUC-ROC
